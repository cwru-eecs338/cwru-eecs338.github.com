02/22/11: Monitors in Java

Look at how monitors are implemented in Java, with example code for the dining philosophers problem.

02/15/11: System V Shared Memory

Introduce API for shared memory in System V Unix, using the same example code for the bounded buffer problem.

02/08/11: System V Semaphores

Introduce API for semaphores in System V Unix, with delicious example code for the bounded buffer problem.

02/01/11: Structs and Pipes

Examples of using structured types in C, followed by an introduction to unnamed pipes.

01/25/11: C, System Calls, and You

The second recitation covers some basics about the C programming language that you might not have learned (or have forgotten since you took Compilers). This semester, you can use Java for your assignments, but C is recommended.

In addition to C basics, various system calls are reviewed, with examples of fork and the environment variable calls.

The recitation slides are available here, or from the slides page.

An introduction to Linux, which will be used for the programming assignments.
